Modern CMS for web publishing

Installation:

  1. Unzip the package in an empty directory and upload everything.
  2. Open wp-admin/install.php in your browser. It will take you through the process to set up a wp-config.php file with your database connection details.
  3. Once the configuration file is set up, the installer will set up the tables needed for your blog. If there is an error, double check your wp-config.php file, and try again. If it fails again, please open an issue at bug tracker with as much data as you can gather.
  4. The installer should then send you to the login page. Sign in with the email and password you specified during the installation.

Migrating from other systems

calmPress utilizes the same database structure as WordPress and therefor you can use any of the relevant WordPress tools to import from a number of systems. First you need to get WordPress installed and working as described above, before using our import tools.

System Requirements

  • PHP version 5.6.20 or higher.
  • MySQL version 5.0 or higher.

Recommendations

  • PHP version 7.3 or higher.
  • MySQL version 5.6 or higher.
